## Translate button

### Information

Clicking on the current flag will drop down a choice of languages.

### Languages

1. `English` : English
2. `Français` : French
3. `Español` : Spanish
4. `Português` : Portuguese
5. `Deutsch` : German
6. `Italiano` : Italian
7. `Ελληνικά` : Greek
8. `Русский` : Russian
9. `日本語` : Japanese
10. `中文`: Chinese
11. `عربي` : Arabic
